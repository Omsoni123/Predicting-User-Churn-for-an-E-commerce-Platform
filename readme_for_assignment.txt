# User Churn Prediction Project

## Overview
This project predicts user churn for an e-commerce platform using event data. The models used include Logistic Regression and Random Forest, evaluated with metrics like ROC-AUC and accuracy. Interpretability is enhanced using SHAP analysis and feature importance.

## Requirements
Install the following Python libraries to run the notebook:
- pandas
- numpy
- scikit-learn
- shap
- matplotlib

You can install them using:
```bash
pip install pandas numpy scikit-learn shap matplotlib
